# Kick Drop Miner



Kick Drop Miner — это десктопное приложение для автоматизации просмотра трансляций на платформе Kick.com, позволяющее эффективно фармить дропы (награды за просмотр) без необходимости ручного взаимодействия с сайтом. Приложение построено на PyQt6, использует Selenium с undetected-chromedriver для работы с браузером, поддерживает локальное сохранение данных и предлагает интуитивно понятный интерфейс на русском языке.

## Основные возможности

### Автоматизация очереди стримов
*   Добавляйте несколько URL стримов Kick.com, задавайте целевое время просмотра (в минутах, 0 для бесконечного просмотра).
*   Очередь автоматически пропускает оффлайн-каналы и переходит к следующим, обеспечивая непрерывный фарминг.
*   Поддержка паузы и возобновления при изменении статуса стрима (онлайн/оффлайн).
*   **Визуальный прогресс-бар** на каждой карточке стрима для отслеживания времени просмотра.

### Управление дроп-кампаниями
*   Просмотр активных дроп-кампаний через API Kick.com 
*   Локальное сохранение данных о кампаниях.
*   **Сворачиваемые группы** для удобной организации дроп-кампаний.

### Уведомления
*   Асинхронные **Telegram-уведомления** через `aiogram`.
*   Настраиваемые десктопные уведомления (через `plyer`) с иконкой приложения.

### Интерфейс и локализация
*   Современный интерфейс на основе PyQt6 с поддержкой тёмной темы.
*   Полная локализация на русском языке с переводом всех элементов интерфейса.
*   **Неизменяемый размер окна** для оптимального отображения.
*   **Улучшенное отображение шрифтов** и подгонка текста на кнопках.
*   **Двойной клик по карточке** для выбора и последующего редактирования.
*   **"Водяной знак" в статус-баре** со ссылкой на Telegram создателя.

## Требования

*   **Операционная система**: Windows 10/11 (Linux/macOS могут работать, но не тестировались).
*   **Python**: 3.12 (рекомендуется, так как Python 3.13 имеет известные проблемы с текущей версией Nuitka).
*   **Браузер**: Установленный Google Chrome.
*   **Интернет**: Для загрузки `chromedriver` и взаимодействия с API Kick.com.
*   **C-компилятор**: Для сборки приложения с Nuitka (например, Microsoft Visual C++ Build Tools для Windows).

## Использование

### Установка
1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/whyz012/KickDropsMiner.git
    cd KickDropsMiner
    ```
2.  Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```

### Вход и Cookies
1.  Нажмите кнопку "Cookies", чтобы открыть Chrome и войти на Kick.com.
2.  После входа в учетную запись нажмите "ОК" в появившемся диалоге, чтобы сохранить cookies.
3.  Если установлен `browser_cookie3`, cookies могут быть автоматически импортированы из браузера, минимизируя ручной вход.
4.  Cookies сохраняются в `cookies/kick.com.json` (в директории пользовательских данных вашего приложения) и применяются для всех операций.

### Добавление стримов
1.  Нажмите кнопку "Добавить стрим".
2.  Введите URL стрима (например, `https://kick.com/channel`) и укажите желаемое время просмотра (в минутах, 0 для бесконечного просмотра).
3.  Стрим отобразится в виде карточки с информацией о цели, прошедшем времени и статусе.
4.  Используйте кнопку "Удалить" на карточке для удаления стрима (останавливает активный процесс).
5.  Для редактирования стрима **дважды нажмите на карточку**, чтобы выбрать ее, затем нажмите кнопку "Редактировать" в верхней панели, измените URL или время и сохраните.

### Запуск очереди
1.  Нажмите кнопку "Запустить Все", чтобы начать последовательный просмотр стримов.
2.  Оффлайн-стримы автоматически пропускаются, а очередь переходит к следующему.
3.  Статусы: "В ЭФИРЕ", "ПАУЗА" (для оффлайн-стримов), "ЗАВЕРШЕНО", "ОСТАНОВЛЕНО".
4.  Остановите отдельный стрим кнопкой "Стоп" на карточке или всю очередь через кнопку "Остановить Все".

### Дроп-кампании
1.  Нажмите кнопку "Drops", чтобы открыть окно с активными кампаниями.
2.  Просмотрите доступные кампании, каналы и их статус.
3.  Добавьте отдельный канал ("+ Добавить") или все каналы кампании ("Добавить все") в очередь просмотра.
4.  Кампании загружаются через API с использованием сохранённых cookies.

### Управление воспроизведением (Настройки)
В окне "Настройки" (кнопка "Настройки" в главном окне) вы можете настроить:
*   **Отключить звук**: Включает mute для всех стримов.
*   **Скрыть плеер**: Запускает Chrome в headless-режиме, скрывая видео.
*   **Мини-плеер**: Отображает компактное окно поверх других приложений.
    *   *Примечание*: При использовании расширений Chrome headless-режим отключается автоматически.

### Уведомления
1.  Настройте Telegram-уведомления через кнопку "Telegram", указав токен бота и Chat ID.
2.  Получайте уведомления о достижении цели, оффлайн-стримах, ошибках и ключевых словах в чате.
3.  Десктопные уведомления отображаются через `plyer` с иконкой приложения.

### Статистика
1.  Нажмите кнопку "Статистика", чтобы открыть окно с общей статистикой просмотра и временем по каждому стриму.
2.  Нажмите "Открыть лог сессий", чтобы просмотреть файл `session_log.json` в текстовом редакторе.
